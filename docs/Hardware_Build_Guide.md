# ReflowDesk Hardware Build Guide

## ReflowDesk AT-MK1 Controller Board
### Hardware Required
- ReflowDesk AT-MK1 Controller Board PCB rev1.0

## DIY Version
### Hardware Required
- ESP32 Development Board
- I2C or SPI SSD1306 OLED Display Module
- 5VDC, 240VAC/2A Solid State Relay Module (3.3V Logic Compatible)
- Rotary Encoder Module (with Push Button and Pullup resistors)
- MAX6675 Thermocouple Module with K-Type Thermocouple
- ADS1115 16-bit ADC Module
- 100K NTC 3950 Thermistor
- 100K 1% resistor
- 10uF/16V Electrolytic Capacitor
- Buzzer Module (3.3V Logic Compatible)
- WS2812B RGB LED Module (Single Pixel)
- 12VDC 4-Wire PWM Fan (80MM/90MM)
